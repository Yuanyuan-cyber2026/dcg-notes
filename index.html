科目编号规律

1xx = 资本与权益（Capitaux）  2xx = 固定资产（Immobilisations）  3x = 存货（Stocks）

4xx = 第三方往来（Tiers）  5xx = 金融/银行（Financiers）

6xx = 费用（Charges）  7xx = 收入（Produits）

28x = 2x的累计折旧 · 29x = 2x的减值 · 39x = 3x的减值 · 49x = 4x的减值

68x = 折旧/减值/准备的计提（费用）· 78x = 折旧/减值/准备的转回（收入）

🔄 PCG 2025重大变动：675→657、775→757（资产处置从特殊损益移至经常损益）；791/796/797删除；671/771删除
